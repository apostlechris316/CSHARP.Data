# CSHARP.Data
This project provides examples and starting points for various ways to manipulate data. 
These libraries are meant to provide starting points for you and do not necessarily represent final production code.

LICENSE: Free to use provided details on fixes and/or extensions emailed to chris.williams@readwatchcreate.com

References:

	* This library only references standard .NET 4.5 framework dlls.

Libraries: 

	* This project contains 9 C# projects:
	
	CSHARP.Data: This project contains helper classes when working with .NET DataSets. 
	
	CSHARP.Data.Common: This project contains common data object classes such as custom fields, external ids, etc
	
	CSHARP.Data.Json: This project contains helper classes when working with JSON text.
	
	CSHARP.Data.Json.FileSystem: This project contains helper classes to assist in reading and writing JSON items from the file system.	
	
	CSHARP.Data.SqlServer.Management: This project contains helper classes to assist in managing a SQL Server Instance

	CSHARP.Data.SqlServer.MSPractices: This project contains helper classes to assist in working with Microsoft Practices Library to work with SQL Server.

	CSHARP.Data.SqlServer.Standard: This project contains helper classes to assist in working with SQL Server using Standard .NET

	CSHARP.Data.Xml.Atom: This project contains helper classes to assist with manipulating ATOM feeds

	CSHARP.Data.Xml.Rss: This project contains helper classes to assist with manipulating RSS feeds.
